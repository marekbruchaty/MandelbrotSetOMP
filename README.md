# MandelbrotSetOMP
Mandelbrot Set Fractal implemented in C++ with OpenGL &amp; OMP

**Build executable on Mac OS & Linux**
``` shell
gcc-5  -framework OpenGL -framework GLUT -lobjc -fopenmp ./MandelbrotSetOMP.cpp
```
